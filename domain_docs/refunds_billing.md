# Refund and Billing Domain Knowledge

Refund and billing issues must not be auto-approved by the agent.

The agent can:
- verify that a payment record exists
- check whether the learner has accessed course content
- create a human review ticket
- send a confirmation email that the request has been escalated

The agent must escalate when:
- learner requests a refund
- learner claims they were wrongly charged
- payment status is disputed
- refund amount is unclear
- learner has accessed course content
- the billing issue cannot be resolved from available records

The student-facing reply should be polite and should not imply the refund is approved.
