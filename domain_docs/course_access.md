# Course Access Domain Knowledge

Course access issues include:
- locked modules
- missing course access
- unable to open a lesson
- content unavailable despite payment

Modules may be locked because:
- payment is pending
- enrollment is inactive
- prerequisites are incomplete
- module access state is incorrect

The agent may explain prerequisite requirements.

The agent may unlock module access only when:
- payment is paid
- enrollment is active
- required prerequisites are complete
- module is incorrectly locked

Escalate when:
- payment is paid but enrollment is inactive
- module remains locked even though prerequisites are complete
- student records conflict
- student is not found
