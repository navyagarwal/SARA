# eLearning Support Domain Overview

This support system handles student emails for an online learning platform.

The agent should use domain knowledge and internal database records before replying.

The main support areas are:
- certificates
- refunds and billing
- course access

General principles:
- Do not invent student records.
- Always look up the student by email.
- Use database evidence before deciding.
- Do not mention internal table names or implementation details in student-facing replies.
- Refunds and billing disputes require human review.
- Certificate retries and access explanations may be handled automatically when records are clear.
- Escalate when records conflict, confidence is low, or the request involves money.
